모두의 쉐어  발급받기

모두의쉐어는 블로그/티스토리/홈페이지 등 다양한 사이트에서 이용이 가능한 서비스입니다.
쉐어를 1회 발급받으면 배너이미지, 설정 등의 손쉬운 변경이 가능합니다.
  • step.01 이용약관 동의
  • step.02 쉐어 코드 발급
  • step.03 쉐어 코드 확인

쉐어 발급 방법

1. 아래의 ‘쉐어 코드 발급’ 버튼을 클릭하여 코드를 복사합니다.
2. 쉐어를 설치할 블로그/티스토리/홈페이지 등의 스킨 설정 페이지에 접속합니다.
3. 블로그위젯,홈페이지 스킨등에 쉐어를 추가할 수 있도록 코드를 붙여넣기합니다.
4. 쉐어 발급이 완료된 블로그/티스토리/홈페이지에 접속하시면, 쉐어를 확인하실 수 있습니다.

유의사항

- 음란, 도박 등 부적절한 콘텐츠로 공유하여 해당 제품의 이미지를 실추시키거나 부당한 방법으로 포인트를 취득 할 시 법적 조치 될 수 있습니다.
- 랜덤쉐어는 페이지당 최대 1개 설치할 수 있으며, 2개 이상 설치할 경우 중복설치로 인해 포인트 차감이 발생할 수 있습니다.
- 고정 쉐어는 페이지당 다수 설치할 수 있으며, 하나의 페이지에 설치된 모든 고정쉐어의 포인트가 인정됩니다.
- 부정 카운트 증가행위가 발견 될 시 포인트는 지급되지 않으며 지급된 포인트도 차감 될 수 있습니다.
- 활동 포인트는 실시간으로 누적되며 내 포인트 내역에 실시간으로 기록됩니다.
- 업체 사정으로 인해 상품이 변경 될 수 있습니다.

쉐어 코드 확인 및 설치

1. 블로그/티스토리/홈페이지에 접속하여 블로그 위젯, 홈페이지 관리등을 통해 쉐어를 설치합니다.
2. 쉐어가 정상적인 설치가 되어 있는지 확인합니다.
- 하단의 '내 사이트 바로가기' 클릭 시 회원정보에 기입된 블로그/티스토리/홈페이지 주소로 이동됩니다.
- 회원정보에 기입되지 않은 블로그/티스토리/홈페이지에서 쉐어 서비스 이용을 희망할 경우 'Step.02 쉐어 코드 발급'단계에서 발급받은 코드를 해당 블로그/티스토리로 직접 붙여넣기 바랍니다.
- 추가적인 쉐어 발급을 희망할 시 해당 단계를 한번 더 수행하시기 바랍니다.
